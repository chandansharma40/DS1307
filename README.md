# DS1307
Interfacing DS1307 (RTC) with AVR. A C code to interface RTC with Atmega Microcontrollers.

# Tested on
  1. Atmega1284p
  2. Atmega16
  
# Libraries included
  1. DSB1820.h - 1-wire temperature sensor, values acquired by bit banging
  2. i2cmaster.h - A basic TWI library for Atmega microcontrollers
